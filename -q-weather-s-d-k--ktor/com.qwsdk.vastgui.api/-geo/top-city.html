suspend fun topCity(range: QWSdk.CountryCode = QWSdk.CountryCode.CN, number: Int = 10, lang: QWSdk.Lang = QWSdk.Lang.ZH): Result<GeoTop>

热门城市查询

获取全球各国热门城市列表。

Parameters

range

搜索范围，可设定只在某个国家或地区范围内进行搜索，国家和地区名称需使用 ISO 3166 所定义的国家代码 。如果不设置此参数，搜索范围将在 所有城市。例如 range=cn 。

number

返回结果的数量，取值范围1-20，默认返回10个结果。

lang

多语言设置，请阅读 多语言 文档，了解我们的多语言是如何工作、如何设置以及数据是否支持多语言。

Throws

IllegalStateException

当 number 未在 0-20 范围内会抛出该异常。
