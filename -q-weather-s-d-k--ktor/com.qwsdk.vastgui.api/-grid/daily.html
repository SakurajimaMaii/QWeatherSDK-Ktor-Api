suspend fun daily(days: Day, location: Coordinate, unit: QWSdk.Units = QWSdk.Units.M, lang: QWSdk.Lang = QWSdk.Lang.ZH): Result<GridDaily>

格点每日天气预报

基于全球任意坐标的高精度每日天气预报，精确到 3-5 公里范围，包括温度、湿度、大气压、天气状况、风力、风向等。

Parameters

location

需要查询地区的以英文逗号分隔的经度,纬度坐标 Coordinate 。

unit

数据单位设置，可选值包括unit=m（公制单位，默认）和unit=i（英制单位）。更多选项和说明参考 度量衡单位 。

lang

多语言设置，请阅读 多语言 文档，了解我们的多语言是如何工作、如何设置以及数据是否支持多语言。
