poiLookup

suspend fun poiLookup(location: Location, type: QWSdk.POIType, city: String? = null, number: Int = 10, lang: QWSdk.Lang = QWSdk.Lang.ZH): Result<GeoPoi>(source)

POI搜索

使用关键字和坐标查询 POI 信息（景点、火车站、飞机场、港口等）。

Parameters

location

需要查询地区的名称 Name ，支持文字、以英文逗号分隔的经度，纬度坐标 Coordinate 、LocationID 或 Adcode（仅限中国城市) 。

type

POI类型，可选择搜索某一类型的 POI 。

city

选择 POI 所在城市，可设定只搜索在特定城市内的 POI 信息。城市名称可以是 Name 或城市的 LocationID 。默认不限制特定城市。

number

返回结果的数量，取值范围1-20，默认返回10个结果。

lang

多语言设置，请阅读 多语言 文档，了解我们的多语言是如何工作、如何设置以及数据是否支持多语言。

Throws

IllegalStateException

当 number 未在 0-20 范围内会抛出该异常。
